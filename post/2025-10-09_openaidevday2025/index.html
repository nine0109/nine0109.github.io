OpenAI DevDay 2025

OpenAI DevDay 2025 총정리 — ChatGPT 플랫폼 진화

원본 기사: OpenAI DevDay 2025에서 공개한 모든 것들 | GeekNews

OpenAI가 2025년 DevDay에서 발표한 여러 신기능과 도구들을 정리한 내용입니다.

주요 발표 요약

  • ChatGPT 안에서 앱 실행 기능이 발표되었으며, Spotify, Canva, Coursera, Figma 등 다양한 파트너 앱이 처음 적용 대상입니다. 사용자는 대화 중 필요한 앱을 자연어로 바로 실행할 수 있도록 설계되었습니다.
  • Apps SDK를 통해 개발자들이 ChatGPT용 앱을 직접 제작·테스트할 수 있는 표준 개발 키트가 공개되었습니다.
  • AgentKit이라는 도구 세트로 에이전트(자동화된 AI 도우미)를 개발·배포·평가하는 전용 환경이 제공됩니다. 드래그 앤 드롭 방식의 Agent Builder가 포함되어 빠른 개발이 가능합니다.
  • Codex 플랫폼이 정식 출시되었으며, Slack 통합 및 SDK를 통해 자동 코드 생성과 협업 기능이 강화되었습니다.
  • Sora 비디오 생성 API 등 멀티미디어 생성 도구와 GPT-5 Pro 모델 등 AI 모델 확장 기능들도 발표되었습니다.

Figma 관련 내용

기사에서는 Figma가 OpenAI의 새로운 앱 파트너로 포함되었다는 점이 언급되어 주목을 받았습니다.

이처럼 OpenAI는 ChatGPT를 단순한 챗봇이 아닌 앱 플랫폼으로 확장하고 있으며, 개발 도구부터 자동화 에이전트, 앱 통합까지 전방위 기능 강화를 추진하고 있습니다.
