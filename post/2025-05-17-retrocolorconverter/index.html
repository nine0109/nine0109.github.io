Retro Color Converter 업데이트

들어가며

두 번째 피그마 플러그인 피그마 커뮤니티에 많은 업데이트가 있었습니다. 플러그인은 12bit 디더링 이미지를 빠르게 확인하기 위해서 개발을 시작했습니다. 이후 확장성을 고려해 Rasterized 이미지를 변경하는 방법을 넣다보니 본 기능보다 더 많은 기능을 갖게되었습니다. 개인적으로는 속도가 느려지고 복잡도가 높아졌지만, 다른 누군가에는 조금이나마 도움이 될 수 있길 바라며 기능을 소개합니다.

플러그인 기능

3가지 컬러 댑스 6비트, 9비트, 12비트 또는 사용자 정의 컬러 깊이와 5가지 이미치 처리 모드를 사용해 이미지를 생성합니다. ** 5가지 프로세싱 모드:**

  • Flat Shading: 픽셀화된 이미지와 직교 윤곽선 생성
  • Retro Pixel Art: 제한된 색상 팔레트, 픽셀 윤곽선 및 디더링 효과가 있는 고급 레트로 픽셀 아트 스타일
  • Dot Pattern: 이미지를 조절 가능한 크기, 간격, 스타일 및 배경의 도트 패턴으로 변환
  • Pixel Mosaic: 커스터마이징 가능한 윤곽선이 있는 모자이크 스타일 픽셀화 이미지 생성
  • Gradient Dithering: 그라데이션 경계에만 디더링 적용
  • Full Dithering: 전체 이미지에 Floyd-Steinberg 디더링 적용

모드별 옵션

Flat Shading

  • Outline Thickness: 윤곽선 두께 조절 (1–5 픽셀)
  • Outline Style: 스무스 컬러 윤곽선 또는 고대비 윤곽선 중 선택

Retro Pixel Art

  • Pixel Size: 픽셀 크기 조절 (2–12 픽셀)
  • Use Limited Palette: 16색 레트로 팔레트 적용 여부
  • Use Pixel Outlines: 색상 간 경계에 윤곽선 적용 여부
  • Use Pixel Dithering: 전환을 개선하기 위한 미묘한 디더링 적용 여부
  • Outline Style: 스무스 컬러 윤곽선 또는 고대비 윤곽선 중 선택

Dot Pattern

  • Dot Size: 도트 크기 조절 (2–30 픽셀)
  • Dot Spacing: 도트 간 간격 조절 (0–20 픽셀)
  • Dot Pattern Style: 둥근 도트 또는 사각 도트 중 선택
  • Background: 흰색, 검은색 또는 투명 배경 중 선택

Pixel Mosaic

  • Pixel Size: 모자이크 픽셀 크기 조절 (2–12 픽셀)
  • Outline Thickness: 모자이크 윤곽선 두께 조절 (1–3 픽셀)
  • Outline Style: 스무스 컬러 윤곽선 또는 고대비 윤곽선 중 선택
