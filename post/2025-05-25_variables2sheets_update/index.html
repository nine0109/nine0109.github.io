Variables2Sheets 메이저 업데이트

Figma Variables를 외부로 내보내고 관리하는 플러그인 variables2sheets에 중요한 업데이트가 적용되었습니다.

🔗 Figma Community에서 확인하기

Variables2Sheets 🌟

다국어 프로젝트에서 클라이언트와의 원활한 소통을 위해 디자인 시스템을 개발 후 Figma에 적용한 Variables의 String type을 내보내기 위해 개발한 Variables2Sheets 플러그인을 이제 디자인 시스템을 위한 도구로 사용할 수 있게 업데이트했습니다.

[!note] ** Variables2Sheets의 주요 업데이트**

  1. 전체 Variable 타입 지원
  1. 별칭(Alias) 지원
  2. Variables의 컬렉션 변경
  3. 한국어 지원
  4. 개선된 사용자 경험

전체 Variable 타입 지원 ✅

이제 Figma Variables의 4가지 타입을 모두 지원합니다

기존 MVP 버전에서는 String 타입만 지원했지만, 이번 업데이트로 완전한 Variables 관리가 가능해졌습니다:

  • String - 텍스트 콘텐츠 및 다국어 관리
  • Color - 컬러 시스템 및 팔레트 관리
  • Number - 간격, 크기 및 숫자 값
  • Boolean - 상태 관리 (true/false)

별칭(Alias) 지원 🔗

복잡한 Variable 참조 관계도 완벽하게 관리

디자인 시스템의 핵심인 Variables 간 참조 관계를 스프레드시트에서도 그대로 유지합니다:

  • String, Color, Number, Boolean Alias 지원
  • 복잡한 Variable 참조 관계도 손쉽게 관리
  • 고급 기능을 사용한 실제 값 변환 표시
  • 복잡한 디자인 토큰 시스템의 일관성 보장

고급 컬렉션 관리 🔄

Figma Variables 패널에서는 할 수 없는 컬렉션 간 Variables 이동을 할 수 있습니다.

  • 스프레드시트에서 컬렉션 간 Variables 직접 이동
  • 존재하지 않는 컬렉션명 입력 시 자동 생성
  • 디자인 시스템 구조를 손쉽게 재정리
  • 대규모 Variable 구조 개편이 간단해집니다

한국어 지원 추가 🌐

로컬 사용자를 위한 다국어 인터페이스

2025년 5월 Figma Config에서 발표된 Figma Site처럼, Figma가 CMS 기능으로 확장되고 있습니다. Variables2Sheets로 디자인관리 없이 시트만으로 컨텐츠를 업데이트할 수 있습니다.

  • 사이트 운영자가 Figma 수정 없이 시트만으로 콘텐츠 업데이트
  • 채널 담당자의 편의를 위한 한국어 인터페이스 제공
  • 추가 언어 지원도 곧 업데이트 예정

향상된 사용자 경험 🎨

생산성을 위한 세심한 개선사항

  • 시스템 테마 지원 (다크/라이트 모드)
  • 컬렉션 자동 새로고침으로 실시간 동기화
  • 생산성을 위한 간소화된 인터페이스
  • 쉬운 사용: 클립보드 복사&붙여넣기
  • 파일관리: CSV/JSON 내보내기

업데이트의 의미 💡

From MVP to Complete Solution

초기 MVP는 특정 용도(다국어 String 관리)에 특화되어 있었지만, 이번 업데이트로 완전한 Variables 관리 솔루션으로 발전했습니다.

Future-Ready Design

variables2sheets는 작은 플러그인이지만 Figma Site의 CMS 기능 확장에 발맞춰, 디자인과 콘텐츠 관리의 경계를 허무는 도구로 진화하고 있습니다.

이번 업데이트로 더욱 강력해진 variables2sheets를 경험해보세요!
